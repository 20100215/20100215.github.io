ANYQUIZ is a fully featured quiz maker for school, business, or just for fun. Create one now and explore its awesome features that match your desire and purpose. This is Wayne's very first full stack web application. Originally being a a project for Web Development 1 class, this app has been scaled during summer to implement several more features including varying types of test questions, scoring, and moderations.
