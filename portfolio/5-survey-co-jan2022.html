SURVEY CO. - Likert Scale Survey Management System

Project information

  • Category: Web Application
  • Project Date: January 2022
  • Web Frameworks: Vue.JS and Laravel
  • Project Repository: [Click here]

SURVEY CO. is a web application built using Vue.JS and Laravel that enables businesses to create surveys for their customers to respond by simply inputting the survey code generated and their email addresses. Additionally, it includes data visualization of the obtained results for easier inprepretation by the survey creators.
