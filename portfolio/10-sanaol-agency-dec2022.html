SANAOL AGENCY | Talent Management System

Project information

  • Category: Web Application
  • Project Date: December 2022
  • Tech Stack (Development): Vue, Laravel, TailwindCSS, MySQL
  • Tools for Testing: Selenium, RobotFramework
  • Project Repository: [Click here]

Sanaol Agency is a Web-based content management system appllication designed for talent agencies, hosting a collection of users, services, evemts, and news. Visitors may view the website to see the currently featured events, services, and news through the landing page which are all editable by admins. Talents may edit their profile and view their upcoming event reminders. Admins can approve and deny new talent applications as well as edit and delete existing user info.
